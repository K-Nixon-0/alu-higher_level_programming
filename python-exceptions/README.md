# Python - Exceptions

This project covers exception handling in Python, including `try`, `except`, and `finally` blocks, as well as raising standard exceptions.

## Files
* **0-safe_print_list.py**: Prints x elements of a list safely.
* **1-safe_print_integer.py**: Prints an integer safely.
* **2-safe_print_list_integers.py**: Prints only integers from a list.
* **3-safe_print_division.py**: Divides two integers and prints the result safely.
* **4-list_division.py**: Divides elements of two lists.
* **5-raise_exception.py**: Raises a type exception.
* **6-raise_exception_msg.py**: Raises a name exception with a custom message.
