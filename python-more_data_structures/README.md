# ALU Higher Level Programming - Python

This repository contains Python programming assignments focused on data structures and algorithms at ALU.

## Directories and Files

### 1. [python-data-structures](./python-data-structures)
* **3-print_reversed_list_integer.py**: Prints integers of a list in reverse order using `str.format()`.
* **9-max_integer.py**: Finds the biggest integer in a list without using built-in `max()`.
* **10-divisible_by_2.py**: Finds all multiples of 2 in a list and returns a boolean list.
* **11-delete_at.py**: Deletes an item at a specific position in a list without using `pop()`.
* **12-switch.py**: Switches the values of two variables.

### 2. [python-more_data_structures](./python-more_data_structures)
* **0-square_matrix_simple.py**: Computes the square value of all integers of a 2D matrix.
